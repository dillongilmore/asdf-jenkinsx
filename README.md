# asdf-jenkinsx

[JenkinsX](https://jenkins-x.io/) (jx) plugin for the
[asdf](https://github.com/asdf-vm/asdf) version manager.

## Install

```
asdf plugin-add jenkinsx https://github.com/dillongilmore/asdf-jenkinsx
```
